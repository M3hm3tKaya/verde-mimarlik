# Project Brief - Verde Mimarlık

## Genel Bakış
Verde Mimarlık, sürdürülebilir mimarlık ofisinin 7 sayfalık kurumsal web sitesi. Minimal, yeşil tonlar, doğal estetik.

## Hedef
- Sürdürülebilir mimarlık vizyonunu yansıtmak
- Proje portföyünü sergilemek
- Potansiyel müşterilerle iletişim kurmak

## Teknik Yığın
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- GSAP + Framer Motion + Lenis
- DM Sans font

## 7 Sayfa
1. Ana Sayfa
2. Projeler
3. Proje Detay (dinamik)
4. Hizmetler
5. Hakkımızda
6. Sürdürülebilirlik
7. İletişim
